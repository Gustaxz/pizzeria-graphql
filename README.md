# Pizzeria Graphql

## Descrição

App inspirado no desafio backend [Pizzeria](https://github.com/AmbulnzLLC/fullstack-challenge/tree/master/backend).
Esta aplicação usa GraphQL ao invés do padrão REST do desafio original.

<br>

## Tecnologias usadas:

- Prisma
- Apollo Server
- GraphQL
- Nest JS
- TypeScript

<br>

## Como rodar

```bash
  $ npm run start:dev
```
